## âš™ï¸ Working Principle
The LM3914 IC compares the input battery voltage with internally set reference voltages.  
Based on the input voltage level, the IC drives the corresponding LEDs in sequence. As the battery voltage increases, more LEDs glow, providing a clear visual indication of the battery level.

A potentiometer is used to adjust the voltage range and sensitivity of the indicator.
